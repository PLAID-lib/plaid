PLAID (Physics Learning AI Datamodel) is a library proposing an implementation for a datamodel tailored for AI and ML learning of physics problems.
It has been developed at SafranTech, the research center of [Safran group](https://www.safran-group.com/).

The code is hosted on [GitHub](https://github.com/PLAID-lib/plaid).
