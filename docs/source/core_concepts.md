# Core concepts

PLAID is a datamodel and library for organizing physics datasets and defining learning problems on these datasets.

It provides high-level classes such as [`plaid.containers.dataset.Dataset`](api_reference.md#api-dataset), [`plaid.containers.sample.Sample`](api_reference.md#api-sample), and [`plaid.problem_definition.ProblemDefinition`](api_reference.md#api-problemdefinition).

PLAID relies on the CGNS standard for representing complex physics meshes and uses human-readable formats like `.yaml` and `.csv` for other features.

For more details and examples, see [Core concepts](core_concepts/sample.md) and [Examples and tutorials](examples_tutorials.md).
