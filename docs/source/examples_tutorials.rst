Examples and Tutorials
======================

You can find here detailed examples for different parts of plaid, explained mainly in jupyter notebooks.

.. warning::
   Some historical notebook folders may still exist in the documentation source tree,
   but only the entries listed below are considered maintained references.

.. note::
   The `Muscat=2.5.1 <https://anaconda.org/conda-forge/muscat>`_ dependency is required on some of the notebooks,
   and is not distributed with the `plaid <https://anaconda.org/conda-forge/plaid>`_
   conda-forge package.
